
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
npm install
```

## Running the app

```bash

# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod

# Docker Command

docker-compose up

POST http://localhost:3000/food-menu
{
  "name" : "Pizza",
  "price" : 235
}


```
